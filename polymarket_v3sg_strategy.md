# Polymarket V3-SG "跟单猎手" 策略 v2

## 核心理念
从排行榜出发，通过多维度交叉验证，识别【长期稳定盈利】账号，
排除运气型、做市商型、单日暴富型，只跟真正有边际优势的账号。

## 数据源（全部公开 API）
- 每日榜单：抓取 https://polymarket.com/zh/leaderboard 页面
- 交易历史：https://data-api.polymarket.com/activity?user=<addr>&limit=500
- 交易明细：https://data-api.polymarket.com/trades?user=<addr>&limit=500
- 当前持仓：https://data-api.polymarket.com/positions?user=<addr>&sizeThreshold=0
- 账号价值：https://data-api.polymarket.com/value?user=<addr>

## 多维度评分体系（满分100分）

### 维度1：连续上榜天数（25分）
| 连续天数 | 得分 |
|---------|------|
| 1-2天   | 5    |
| 3-4天   | 10   |
| 5-6天   | 18   |
| 7天以上  | 25   |
亏损天数 > 1天 → 直接淘汰

### 维度2：交易规律性（25分）——机器人识别核心
分析最近500条交易记录：
- 下注时间间隔标准差 < 60秒 → 25分（高度规律，极可能是机器人）
- 标准差 60-300秒 → 18分
- 标准差 300-1800秒 → 10分
- 标准差 > 1800秒 → 5分（人类随机）

金额变异系数 < 0.3 → 额外 +5分（bonus，固定金额下注）

### 维度3：历史胜率（25分）
分析近500笔已结算交易的盈亏：
- 胜率 ≥ 70% → 25分
- 胜率 60-70% → 18分
- 胜率 55-60% → 12分
- 胜率 50-55% → 6分
- 胜率 < 50% → 0分（淘汰）

### 维度4：品类专注度（15分）
- 高度专注1-2个品类（如只做体育）→ 15分（有垂直优势）
- 3-4个品类 → 10分
- 5+个品类 → 5分（广撒网型）

### 维度5：资金规模趋势（10分）
对比 value API 和历史投入总额：
- 账号总价值持续增长 → 10分
- 持平 → 5分
- 下降 → 0分

---

## 综合门槛
- **总分 ≥ 60分** → 进入跟单名单
- **总分 ≥ 80分** → 高优先级，优先下注
- 每日重新评分，动态增补/剔除

---

## 每日管理流程
```
每日 17:00 UTC（榜单刷新后）：
1. 抓取当日榜单存档
2. 对榜单前50名逐一评分
3. 高分账号加入/保留跟单名单
4. 低分/消失账号从名单移除
5. 记录变更日志到 v3sg_accounts.json
```

---

## 跟单执行规则
- 扫描跟单名单中所有账号的**当前活跃持仓**
- 过滤：价格0.10-0.90，结算30min-7天
- 按账号评分 × 仓位大小 排序
- 下注金额：
  | 账号评分 | 下注 |
  |---------|------|
  | 60-70分 | $2   |
  | 70-80分 | $3   |
  | 80-90分 | $5   |
  | 90+分   | $8   |
- 同一市场多个高分账号共同持有 → 翻倍下注

## 与V2-SG的区别
| | V2-SG | V3-SG |
|--|-------|-------|
| 来源 | 今日榜单Top15 | 历史验证账号名单 |
| 可信度 | 今日表现 | 长期验证 |
| 风险 | 较高（可能运气） | 较低（历史证明） |
| 覆盖面 | 广 | 精准 |
| 启动 | 立即 | 积累7天后满血 |
